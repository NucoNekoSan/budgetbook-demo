# UBIQUITOUS.md - BudgetBook Domain Language

用語の揺れを防ぐための定義。

| 用語 | 定義 |
|---|---|
| 口座 | 現金、普通預金など、お金を保管する単位。`Account` |
| 初期残高 | BudgetBook で管理を始める時点の口座残高。`opening_balance` |
| カテゴリ | 収入・支出の分類。`Category` |
| 収入 | 給与、副収入など資産が増える通常取引 |
| 支出 | 食費、日用品など資産が減る通常取引 |
| 通常取引 | `Transaction`。収入または支出 |
| 振替 | 口座間のお金の移動。`Transfer`。収入・支出には含めない |
| 月初繰越 | 対象月の前日までの全口座合計残高 |
| 当月収支 | 対象月の収入合計 - 支出合計 |
| 月末残高 | 対象月末時点の全口座合計残高 |
| 支出構成 | 支出合計を母数にしたカテゴリ別割合 |
| 収入比率 | 収入合計を母数にした支出カテゴリ別割合 |
| 支出カテゴリグループ | 複数の支出カテゴリを分析用にまとめる表示上の分類 |
| 食品・日用品 | スーパー、コンビニ、Co-op、ドラッグストアなどをまとめる想定グループ |
| 無効化 | 過去データを残したまま、新規入力候補から外すこと |
| 実運用 DB | `budgetbook/db.sqlite3`。個人の金融情報を含むため機密扱い |
