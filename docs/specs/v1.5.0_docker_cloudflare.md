# v1.5.0 Docker And Cloudflare

Status: 実装完了

## 目的

Ubuntu Server + Docker + Cloudflare Tunnel + Cloudflare Access で、世帯メンバー（最大 2 名想定）だけが外出先から BudgetBook にアクセスできるようにする。

## 推奨構成

```text
外出先ブラウザ
-> Cloudflare Access
-> Cloudflare Tunnel
-> 自宅 Ubuntu Server
-> Docker
-> BudgetBook
```

## セキュリティ方針

- 自宅ルーターの 80/443 ポート開放はしない。
- Cloudflare Access を Django ログインの前段に置く。
- Django ログインも残し、二重認証にする。
- Cloudflare Access の許可対象は世帯メンバーのメールアドレスのみ。
- Cloudflare Tunnel token は Git 管理しない。

## 実装対象

- `Dockerfile`
- `docker-compose.yml`
- `.dockerignore`
- `requirements.txt` への `gunicorn` 追加
- `.env.example` の本番運用例追記
- Django settings の本番運用設定整理
- `docs/` への Ubuntu Server 運用手順追加

## 受け入れ条件

- `docker compose config` が通る。
- `docker compose build` が通る。
- `DEBUG=False` で `collectstatic` できる。
- `python manage.py test ledger` が通る。
- 秘密情報が Git に入らない。
- Cloudflare 側の Tunnel / Access 設定手順が docs に明記されている。
