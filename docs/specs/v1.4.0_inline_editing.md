# v1.4.0 Inline Editing

Status: 実装完了

## 目的

取引一覧から編集する際、右側の新規登録フォームへ移動する UX を改善する。

## 仕様

- 取引一覧の「編集」押下時、対象行の直下に編集フォームを展開する。
- 右側フォームは新規登録専用として維持する。
- キャンセルでインライン編集フォームを閉じる。
- 保存後は対象月の一覧、集計、グラフ、口座残高を更新する。
- HTMX を使い、ページ全体の再読み込みは避ける。
- スマホ表示でも編集フォームが崩れないようにする。

## 受け入れ条件

- 編集対象の文脈を保ったまま修正できる。
- 保存後に集計が即時反映される。
- 既存の新規登録フォームが壊れない。
- `python manage.py test ledger` が通る。
