# v1.13.0 確定申告レポート（税控除タグ集計）

## 目的

`Category.tax_tag` (medical / donation / business / other) でタグ付けされた支出取引を年単位で集計・閲覧・CSV ダウンロードできるレポート機能を追加する。確定申告期（2-3 月）に医療費控除等の集計を自動化する。

## 設計（合意済み）

- **カバー範囲**: 全 tax_tag 汎用（medical / donation / business / other）
- **出力**: HTML プレビュー（テーブル + 合計） + CSV ダウンロード
- **動線**: `/settings/` の button-row に「📑 税控除レポート」追加

## スコープ

### URL / view

- `GET /reports/tax-deductions/?year=YYYY&tax_tag=medical` — HTML 表示
- `GET /reports/tax-deductions.csv?year=YYYY&tax_tag=medical` — CSV ダウンロード
- どちらも `@login_required`、`@require_http_methods(['GET'])`

### クエリパラメータ

| 名前 | 必須 | デフォルト | 値 |
|---|---|---|---|
| `year` | × | 現在年 | 整数 |
| `tax_tag` | × | `medical` | `medical` / `donation` / `business` / `other` |

無効な値はデフォルトにフォールバック。

### 集計対象

```python
Transaction.objects.filter(
    date__year=year,
    category__tax_tag=tax_tag,
    category__kind=Category.Kind.EXPENSE,  # 控除対象は支出のみ
)
```

支出のみを対象。Transfer / 収入は対象外。

### HTML 表示内容

- ヘッダー: 「2025 年 医療費控除レポート」
- 年・タグ切替セレクト（フォーム）
- サマリーカード:
  - 件数
  - 合計金額
  - 医療費控除の場合は「あと ¥X で控除ライン突破」ヒント（10 万円 - 合計）
- 取引明細テーブル:
  - 列: 日付 / 支払先(摘要) / カテゴリ / 金額 / メモ
  - 日付昇順
- 該当 0 件時の空状態メッセージ
- CSV ダウンロードボタン

### CSV 仕様

- 文字コード: UTF-8 BOM 付き（Excel 文字化け回避、既存 export と同じ方針）
- ヘッダー: `日付,支払先,カテゴリ,金額,メモ`
- ファイル名: `tax-{tag}-{year}.csv` 例: `tax-medical-2025.csv`
- CSV injection 対策: `csv_safe_row` 経由で出力（既存 utility を流用）

### 動線

- `/settings/` の button-row に「📑 税控除レポート」リンク追加（ログイン履歴の隣）

### 事前準備（ユーザー側）

各カテゴリの `tax_tag` を Web UI で設定する。デフォルトは `none` なので、設定しないと 0 件レポートになる。設定方法は `/settings/` の カテゴリ編集画面（既存）から `税控除タグ` を選択。

## DB 影響

- **migration なし**（`tax_tag` フィールドは既存）
- データ変換なし
- 既存データは tax_tag=none のまま、新規レポートで自動的に「該当 0 件」と表示される

## セキュリティ / データ保護

- `@login_required` で前段ガード（既存運用と同じ）
- CSV injection 対策: `csv_safe_row` で `=`, `+`, `-`, `@` 始まりのセルにシングルクオートを前置
- 入力値 (year / tax_tag) は Django の `int()` / 選択肢検証で sanitize
- 個人金融データを含む CSV はブラウザで直接ダウンロード（外部送信なし）

## 想定運用フロー（年末〜申告期）

```
1. 毎年 1 月 / 12 月: 一年使ったカテゴリの tax_tag を見直す
2. 2-3 月: /reports/tax-deductions/?year=前年&tax_tag=medical
3. 合計を確認、控除ライン（¥100,000）突破していれば申告メリットあり
```
